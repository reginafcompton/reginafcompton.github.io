RANDOM NUMBERS

Ruby

Puts rand(101)

=> prints an integer from 0 to 100

JavaScript 

var randNum = Math.random()
console.log(randNum)

This operation generates a number (integers and floats) between 0 and 1 (not including 0 or 1) 

var randNum = Math.random() * 101
console.log(randNum)

This operation generates a number between 0 and 1, then multiplies that number by 101. Alternatively, the below operation generates an integer. 

var randNum = Math.floor(Math.random() * 101)
console.log(randNum)
