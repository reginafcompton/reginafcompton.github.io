This "cheat sheet" contains brief summaries of several Ruby and JavaScript details – namely, those tiny, little things that do not easily stick in one's head. More updates to come!

Random Numbers

Ruby

Puts rand(101)
=> prints an integer from 0 to 100

JavaScript 

var randNum = Math.random()
console.log(randNum)

This operation generates a number (integers and floats) between 0 and 1 (not including 0 or 1) 

var randNum = Math.random() * 101
console.log(randNum)

This operation generates a number between 0 and 1, then multiplies that number by 101. Alternatively, the below operation generates an integer. 

var randNum = Math.floor(Math.random() * 101)
console.log(randNum)

Minimum and   Maximum Numbers

Ruby

The following operations will return the smallest or largest number of the array. 

p [4, 5, 3, 22, 1].min
=> 1

p [4, 5, 3, 22, 1].max
=> 22

JavaScript

console.log(Math.min(3, 5, 9, 0, 66, 1))
=> 0

console.log(Math.max(3, 5, 9, 0, 66, 1))
=> 66

Sorting

Ruby

The following returns the array sorted in ascending order.

puts [3, 5, 2, 1, 9].sort
=> [1, 2, 3, 5, 9]

JavaScript

The JavaScript sort function can be modified to fit the needs of the program, e.g., an ascending or descending sort method.

function sortAscending (a, b) {
  return a – b;
}

[3, 5, 2, 1, 9].sort(sortAscending)
=> [1, 2, 3, 5, 9]
